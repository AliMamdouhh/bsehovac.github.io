ScrollSnap

Lightweight JavaScript library that helps you build smooth sticky horizontal scrollers for the web.

  • 1.4KB minified and gzipped
  • No dependencies

Getting Started

Download ScrollSnap and and link scrollsnap.min.js in your HTML:

<script src="scrollsnap.min.js"></script>

Required HTML

The HTML is pretty simple. Create wrapper, named scrollsnap, and holder with child items that will scroll horizontally:

<div class="scrollsnap">
  <div class="holder">
    <div class="item"></div>
    <div class="item"></div>
    <div class="item"></div>
    <div class="item"></div>
  </div>
</div>

CSS Options

There are few different ways to arrange these items horizontally in holder, and which path you choose will probably come down to personal preference and/or browser support.

The Nowrap Method

Here’s all the CSS that we need. No vendor prefixes, just some simple use of inline-block and a white-space property nowrap you have probably heard of:

.viewport {
  overflow-x: hidden;
}

.holder {
  display: inline-block;
  white-space: nowrap;
  font-size: 0.1px; /* IE doesn't like 0 */
}

.item {
  display: inline-block;
}

The Flex Method

CSS Flex property can make this work for us too. You might need to pull in some vendor prefixes for older browsers, but at least this solution works with IE and Edge:

.viewport {
  overflow-x: hidden;
}

.holder {
  display: flex;
  flex-wrap: nowrap;
}

.item {
  flex: 0 0 auto;
}

Usage

We will just call ScrollSnap() on our viewport element:

var scrollsnap = new ScrollSnap('.viewport');

That’s it. Only one line of JavaScript code and we’ve got snappable horizontal scroller.

API

Function

ScrollSnap(HTMLElement|String, Options);
  • HTMLElement or String is required.
  • String must be CSS Selector
  • String will get only first element of query.
  • Options are optional.

Examples

// HTMLElement
var viewport = document.querySelector('.viewport');
var scrollsnap = new ScrollSnap(viewport);

// CSS Selector
var scrollsnap = new ScrollSnap('.viewport');

// Options
var scrollsnap = new ScrollSnap('.viewport', {
  start: 0,
  duration: 350,
  easing: 'ease',
  holderClass: 'holder',
  itemClass: 'item',
  overscroll: true,
  autoRefresh: false
});

Options

Name Type Description
start Integer Item to show first on load.
duration Integer Transition duration in milliseconds.
easing String Supports all CSS3 transition-timing-function property values.
holderClass String CSS class of holder element.
itemClass String CSS class of item elements.
overscroll Boolean Enable overscroll if you have more than one items visible in viewport at once.
autoRefresh Boolean Enable autorefresh on window resize if scroller dimensions are changed.

Default values are listed in example abowe.

Methods

refresh()

Recalculate ScrollSnap if scroller dimensions are changed or some items are added or removed.

var scrollsnap = new ScrollSnap('.viewport');
scrollsnap.refresh();

slide()

Return current item.

var scrollsnap = new ScrollSnap('.viewport');
scrollsnap.slide();
// > 0

length()

Return number of items in viewport.

var scrollsnap = new ScrollSnap('.viewport');
scrollsnap.slide();
// > 4

first()

Check if current item is first. Return true or false.

var scrollsnap = new ScrollSnap('.viewport');
if (scrollsnap.first()) {
  // do something
}

last()

Check if current item is last. Return true or false.

var scrollsnap = new ScrollSnap('.viewport');
if (scrollsnap.last()) {
  // do something
}

next(duration)

Move to next item.

var scrollsnap = new ScrollSnap('.viewport');
var next = document.querySelector('.next');
next.onclick = function() {
  scrollsnap.next();
}

prev(duration)

Move to previous item.

var scrollsnap = new ScrollSnap('.viewport');
var prev = document.querySelector('.prev');
prev.onclick = function() {
  scrollsnap.prev();
}

move(item, duration)

Move to desired item.

var scrollsnap = new ScrollSnap('.viewport');
var moveButton = document.querySelector('.moveButton');
var moveItem = document.querySelector('.moveInput').value;
moveButton.onclick = function() {
  var item = moveItem.value;
  scrollsnap.move(item);
}

Callbacks

start(callback)

Fired on scrolling start.

var scrollsnap = new ScrollSnap('.viewport');
scrollsnap.start(function() {
  // do something
});

drag(callback)

Fired while scrolling.

var scrollsnap = new ScrollSnap('.viewport');
scrollsnap.drag(function(e) {
  e.direction;  // > direction value, 1 or -1
  e.delta;      // > delta x value in px
});

end(callback)

Fired on scrolling end.

var scrollsnap = new ScrollSnap('.viewport');
scrollsnap.end(function(e) {
  e.moved;  // > is moved, true or false
  e.old;    // > old item scrolled
  e.new;    // > new item scrolled
});

Compatibility

ScrollSnap works on any JavaScript enabled browser that supports both CSS Transform 3D and CSS Transition. This includes Internet Explorer 10+, and most modern desktop and mobile browsers.
